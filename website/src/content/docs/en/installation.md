# Installation

Get Rekapu up and running in just a few minutes.

## Chrome Web Store Installation

The easiest way to install Rekapu is through the Chrome Web Store:

1. Visit the [Rekapu extension page](https://chrome.google.com/webstore) on the Chrome Web Store
2. Click the **Add to Chrome** button
3. Click **Add extension** when prompted
4. The Rekapu icon will appear in your browser toolbar

## Manual Installation (Development)

If you're installing from source or a release package:

1. Download the latest release from GitHub
2. Extract the ZIP file to a folder on your computer
3. Open Chrome and navigate to `chrome://extensions/`
4. Enable **Developer mode** using the toggle in the top right
5. Click **Load unpacked**
6. Select the extracted folder (or `dist` folder if installing from source)

## Verify Installation

To verify that Rekapu is installed correctly:

1. Look for the Rekapu icon in your browser toolbar
2. Click the icon to open the popup interface
3. You should see the welcome screen

## Troubleshooting

If you encounter issues during installation:

- Make sure you're using Chrome version 88 or higher
- Check that no other blocking extensions are interfering
- See the [Troubleshooting](/docs/troubleshooting) guide for more help
